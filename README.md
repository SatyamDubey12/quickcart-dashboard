# QuickCart | Admin Dashboard 🛒

A modern, responsive e-commerce admin dashboard built with **Vanilla JavaScript**, **Bootstrap 5**, and **FakeStoreAPI**. This project was developed as part of a technical assessment to demonstrate skills in API integration, DOM manipulation, and responsive UI design.

## 🚀 Features

- **Dynamic Data Fetching:** Real-time product data fetched from `fakestoreapi.com`.
- **Search Functionality:** Instant filtering of products by title.
- **Category Filtering:** Dynamically generated category dropdown based on API data.
- **Sorting:** Sort products by price (Low to High & High to Low).
- **Interactive UI:**
  - Smooth hover effects on product cards.
  - Detailed product information via Bootstrap Modals.
  - Real-time "Add to Cart" counter simulation.
- **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop using Bootstrap Grid.

## 🛠️ Tech Stack

- **HTML5** & **CSS3** (Custom transitions and media queries)
- **Bootstrap 5** (Layout and Components)
- **JavaScript (ES6+)** (Async/Await, Fetch API, Array Methods)

## 📂 Project Structure

- `index.html`: Main dashboard structure and UI elements.
- `style.css`: Custom styling, hover animations, and responsiveness.
- `script.js`: API logic, filtering, sorting, and DOM updates.
